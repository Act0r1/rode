# Rode design showcase

This folder contains a self-contained interactive HTML prototype for Rode. It
has no external dependencies and includes five screens:

- ChatGPT login and onboarding
- Coding-agent workspace
- Git changes and inline diff
- Native terminal
- Appearance settings and themes

Open `rode-design.html` directly in any modern browser. The navigation, themes,
login state, composer, commit button, terminal commands, and settings toggles are
interactive presentation controls; they do not change project files.

## Signal Foundry concept

`rode-signal-foundry.html` is a more experimental second direction. It treats
Rode as a precision command center for parallel agents, with an execution rail,
repository heatmap, system telemetry, a spatial command constellation, and a
cinematic identity screen. Double-click the RODE wordmark to cycle its three
color palettes.
